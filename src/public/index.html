Image Abstraction Layer

Receive JSON data from Google's Custom Search API.

Append the search term you would like to use to "https://img-abstract.herokuapp.com/img/".

Example usage: "https://img-abstract.herokuapp.com/img/giraffes"

You can enable pagination by appending the query parameter "offset=" to the url.

Example usage: "https://img-abstract.herokuapp.com/img/giraffes?offset=2"


You can view the 25 most recent searches and their associated times at "https://img-abstract.herokuapp.com/img".
